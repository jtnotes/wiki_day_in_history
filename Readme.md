# Build
## Prerequisite
Install nvm.

Install frontend dependencies:
```sh
cd static
npm i
```
## Build backend
In project root directory:
```sh
npm run backend
```

## Build frontend
In project root directory:
```sh
npm run frontend
```

# Configuration
Prepare a config.yml if customization is needed. Just copy it from the sample file and edit.
```sh
cp config.yml.sample config.yml
```

# Usage
In project root directory:
```sh
./wdih
```
